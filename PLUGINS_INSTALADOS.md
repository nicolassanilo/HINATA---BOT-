## 📝 NOTAS IMPORTANTES

### ⚡ Activación de Plugins

1. **Reinicia el bot** para cargar los nuevos plugins:
   ```bash
   # Detén el bot (Ctrl+C) y ejecuta:
   npm start
   ```

2. **Recarga plugins sin reiniciar** (si el comando existe):
   ```
   .reload
   ```

### 🔧 Configuración Recomendada

1. **Habilita los sistemas anti** que necesites:
   - Antilink para proteger contra spam de grupos
   - Antispam para prevenir flood
   - Antitoxic para filtrar lenguaje ofensivo

2. **Configura los sistemas auto** según tu preferencia:
   - Welcome para dar bienvenida a nuevos miembros
   - Autolevelup para gamificación
   - Autoreaccionar para interacción automática

3. **Prueba los comandos de IA**:
   - `.ia` para chat general
   - `.dalle` para generar imágenes
   - `.gemini` para respuestas de Google

### ⚠️ Advertencias

- Los **plugins de contenido adulto** están activos. Considera deshabilitarlos en grupos públicos.
- Algunos **downloaders** dependen de APIs externas que pueden cambiar.
- Los **sistemas anti** deben configurarse según las necesidades de cada grupo.
